# System Set Up

> 自用

## 使用

* curl

  ```bash
  curl -L -o sysset.sh https://raw.githubusercontent.com/faintx/public/main/sysset.sh && source sysset.sh
  ```

* wget

  ```sh
  wget --no-check-certificate -O sysset.sh https://raw.githubusercontent.com/faintx/public/main/sysset.sh && source sysset.sh
  ```
